  Часто задаваемые вопросы
   
Вопрос: Как сделать screen-glide баннер
Ответ: 

Изготовление SWF-баннеров

"Распахивающийся" банер должен состоять из двух отдельных мувиков
(в отдельных swf-файлах).
В первом, видимом, в мувике типа button (его удобнее разместить в отдельном
слое поверх всех имеющихся слоев) нужно добавить такой код:

on (rollOver) {
getURL("javascript:show();");
}

именно в button, а не во фрейме, в котором он находится. Для этого
достаточно один раз щелкнуть мышкой по мувику. Он может состоять всего из
одного фрейма Hit, причем размеры объекта в этом фрейме должны полностью
"накрывать" весь банер, чтобы он весь был кликабельным.

Во втором мувике, который будет показываться при наведении на банер, в
аналогичном button, лежащем поверх всех слоев, нужно разместить такой код:

on(rollOut) {
getURL("javascript:hide();")
}
on(release) {
getURL(bn_url, "target=_blank");
}

При экспорте обоих мувиков в swf-файлы нужно выключить опцию "Compress
movie" или сохранить их как Flash 5 (если используемый в мувиках другой
экшн-скрипт это позволяет, т.е. при сохранении в младшей версии скрипт (-ы)
останется работоспособным).

Загрузка баннера в систему.

Сам баннер оформляется как .js для загрузки в систему. Образец -
bbn.img.com.ua/screen_glide.js
В файле примера поменять URL swf-баннеров, которые надо разместить на Вашем сервере.
banner_src_small - это путь к 468х60 swf-баннеру
banner_src_big - это раскрывающаяся часть.
big_height - высота раскрывающегося баннера.
bn_url пропишет сервер баннерной системы. А значение возьмёт то, что указано в интерфейсе при добавлении баннера.

Установка кода на сайте

Код для вызова баннера (код, который размещается на сайте) должен быть специальным.

<ScRipT lanGuagE="jAvAscrIpt" src="http://ad0.bigmir.net/t.bbn?xxxxx&x&j&"></script>

 ххх&х& - это первые параметры из кода <iframe src=http://ad0.bigmir.net/t.bbn?15532&0&>

Теперь вы владеете всеми необходимыми древними знаниями. Ждём Ваших баннеров;)
